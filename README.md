## 스터디 진행 방식
- 매 주 토요일 오후 7시에 진행합니다.
- '초보 웹 개발자를 위한 스프링5 프로그래밍 입문' 책으로 진행합니다.
- 발표는 로테이션으로 정해진 사람이 발표를 합니다.
- 매 주 한 챕터 씩 공부한 이후 깔끔하게 정리합니다.
